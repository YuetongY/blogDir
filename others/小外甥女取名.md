# 小外甥女取名

姐姐：周清怡

要求：

1. 沿留 `周清` 两字

2. 要求第三个字和 `怡` 有连贯性，有一定寓意，要好听

## 1. 依据

1. 文论语，武周易；女诗经，男楚辞

2. 怡（形容词）：喜悦；愉快

3. 成语：心旷神怡 ——宋·范仲淹《岳阳楼记》、怡然自得

4. 清怡：清平而和悦

## 2. 忌讳

1. 忌谐音作死：如杨伟、殷静

2. 忌用字生僻：王豖、章茻、章燚、章㵘、章朤、章䨻

3. 忌常见多音：苏还、陈数

4. 忌笔画复杂：穆纛雠

5. 忌不雅联想：吴所谓

## 3. 可选的字

- 悦：`怡` 一字本身带有喜悦和愉快的意思
    1. 怡情悦性：安适愉快的样子。形容高 兴而满足的样子
    2. 怡颜悦色：同和颜悦色，温和的脸色，喜悦的表情（颜、色：脸色、表情）。形容和蔼可亲
    3. 怡悦：喜悦

- 姝：美好的样子；美丽，美好

- 静：清静

- 宁：
    1. 怡宁：安宁
