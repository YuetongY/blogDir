# 抛物线代码

```python
import matplotlib.pyplot as plt
h,v0,g= 3000,200,9.8
    t = 1
    while t > 0 and t < (2*h/g)**(0.5):
        xt = v0 * t
        yt = h - 0.5 * g * (t ** 2)
        print(xt, yt)
        plt.plot(xt, yt, 'ro')
        plt.grid('on')
        plt.axis([0, 5000, 0, h])
        t += 1
    plt.show()
```
