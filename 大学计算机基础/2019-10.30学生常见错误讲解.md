### 3.6 超过递归最大深度

![image.png](https://ws1.sinaimg.cn/large/006alGmrly1g8gep5nxrrj317t0hfada.jpg)

**正确代码**:

```python
def sumInt(n):
    #请在下面编写代码
    # ********** Begin ********** #
    summary = 0
    if n >= 0:
        for i in range(n+1):
            summary += i
    else:
        summary = None
    # ********** End ********** #    
    #请不要修改下面的代码
    return summary
```
