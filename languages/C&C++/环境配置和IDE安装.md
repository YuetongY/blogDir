# 环境配置和 IDE 安装

## 1. ubuntu

### 安装 Clion

**下载地址**：[https://www.jetbrains.com/clion/download/#section=linux](https://www.jetbrains.com/clion/download/#section=linux)

### 安装C++开发环境

```shell
#安装gcc
sudo apt-get install gcc
#安装g++
sudo apt-get install g++
#安装cmake
sudo apt-get install cmake
```
