# 知识点

## 1. clion 添加外部库

当我们用到外部的库的时候，我们便需要添加外部库的头文件所在目录作为头文件查找目录。在 CMakeLists 中添加以下代码即可：

`include_directories(/home/qiubing/桌面/parallelAlgorithm/include)`
